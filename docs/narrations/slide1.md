In this demonstration, we're going to highlight creating and working with personas in AI assistants.

We will use Claude Code to create skills and personas modeled after a software development team.

If you're not already familiar with personas, this will give you an idea of how to organize and structure your context. In a way, our brains are already trained to think working with teams where members have defined roles.

There are a few things worth noting. The first is that this does not have to be for software development like our example is going to focus on.

You could utilize the same techniques if you work in marketing, sales, human resources, or construction — a dentist's office, if you're a student, running your own business, or any area where you could imagine having team coordination to achieve whatever it is you want to achieve.

Another thing worth mentioning is that, although we're going to be using Claude Code, these techniques can be applied to virtually any other AI assistant — whether it's Gemini, ChatGPT, Codex. The approach is the same.

Lastly, I want to make it clear that this is not about replacing people or teams in current organizations. These are personal teams to assist you in your domain as you work with AI agents.

A project manager may have an entirely different concept of a personal assistant team than, say, a software developer. But each could benefit from a project manager or a software developer on their own personal team.
